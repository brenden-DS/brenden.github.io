Projects

Customer Segmentation

This Python project focuses on customer segmentation using the K-Means clustering algorithm. By analyzing customer spending scores and income levels, the project groups customers into distinct segments. The identified clusters provide valuable insights into customer behavior and preferences, enabling the development of targeted marketing strategies and personalized offerings to better serve the diverse customer base.

Supermart Sales Analysis

This Python project analyzes sales trends, customer purchasing behavior, and product performance for a Supermart. Utilizing Pandas for data manipulation and Seaborn/Matplotlib for visualizations, the project explores sales data, identifies popular product categories, and evaluates key performance metrics. The goal is to provide the Supermart with valuable business insights to optimize operations, enhance customer experience, and drive sales growth.

Drug Recommendation System

This Python project develops a drug recommender system that utilizes the Random Forest algorithm to suggest the most suitable medication based on a user's health parameters. The web application, built with Streamlit, allows users to input their details, and the machine learning model, trained on a comprehensive drug and user dataset, provides personalized medication recommendations to assist individuals in making informed healthcare decisions.

SpaceX Launch Success Prediction

This project involves collecting data on SpaceX rocket launches using web scraping and APIs with Python. The data is then cleaned and queried using SQL to prepare it for analysis. Plotly is used to create interactive data visualizations to explore trends and patterns in the launch data. Finally, machine learning techniques are applied to build a model that can predict the success rate of future SpaceX launches based on the historical data. This end-to-end project showcases skills in data collection, data engineering, data analysis, and applied machine learning.

Diabetes Prediction Web App

This Python application, built using the Streamlit framework, allows users to input their health parameters and receive a prediction on their risk of developing diabetes. The app leverages machine learning models to analyze the user's data and provide a personalized assessment. By considering factors such as blood glucose levels, age, BMI, and other relevant health indicators, the model can determine the user's likelihood of having or developing diabetes. This project aims to empower individuals to take proactive steps towards managing their health and preventing the onset of this chronic condition.

Credit Card Fraud Detection

A Python-based credit card fraud detection system that leverages the Random Forest algorithm to classify transactions as either fraudulent or legitimate. The model is trained on a dataset of historical credit card transactions, utilizing various features such as transaction amount, user location, and purchasing patterns to identify suspicious activities.

Adidas Sales Dashboard

An interactive Adidas sales dashboard built in Power BI that visualizes sales trends , regional analysis and product performance, enabling data-driven decision-making for the business.

Walmart Sales Dashboard

Designed an interactive Power BI dashboard to track and analyze Walmart sales performance. It provides insights into sales trends ,product performance and regional variations ,enabling data-driven decision-making for the business growth.

HR Attrition Dashboard

The HR Attrition Dashboard provides comprehensive insights into employee turnover and retention, empowering data-driven decisions to address attrition and maintain a stable workforce.

Customer Churn Analysis

Developed a interactive Power BI dashboard to visualize customer churn. Combined with data modelling ,DAX calculations, and visualization best practices to identify high-risk customers ,reveal trends, and inform retention strategies.
